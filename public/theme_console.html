样式和脚本

使用NBD-Theme请先引用：

样式重置<link rel="stylesheet" type="text/css" href="/stylesheets/reset.css" />
主样式<link rel="stylesheet" type="text/css" href="/stylesheets/style_console.css" />
jQuery
(可选)		<script type="text/javascript" src="http://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.6.2.min.js"></script>
jQuery UI
(可选)		<script type="text/javascript" src="http://ajax.aspnetcdn.com/ajax/jquery.ui/1.8.13/jquery-ui.min.js"></script>
  • 其余jQuery文件请参看：Microsoft Ajax Content Delivery Network
  • 为什么要使用微软提供的支持？请阅读此文：3 reasons why you should let Google host jQuery for you，经过尝试，国内的网站使用微软提供的，速度要更快一些。
